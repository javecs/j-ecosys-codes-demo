# j-ecosys-codes-demo
[Javaの開発環境](https://github.com/javecs/j-ecosys)ができたところで初めに動かしてみるアプリです。

## 使うもの
- Kotlin 1.1
- Spring Boot 1.5.2
- Vaadin 8

## ビルド＆実行
```
$ git clone https://github.com/javecs/j-ecosys-codes-demo.git
$ cd j-ecosys-codes-demo
$ gradle bootRun
```
- [localhost:8080](http://localhost:8080/)にアクセスして見てください。
